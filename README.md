# React Form - Blog Articles

## Descrizione

Applicazione React per gestire una lista di articoli di un blog. Il progetto esplora la gestione dello **state** in React, la gestione degli **eventi** e il rendering condizionale di liste.

---

## Milestone 1 - Lista articoli

Creare una pagina che visualizzi una lista di articoli mostrando solo il titolo di ciascuno.

**Concetti chiave:** `useState`, rendering di liste con `.map()`, uso della prop `key`.

---

## Milestone 2 - Aggiunta articolo

Aggiungere un form con un campo input per inserire il titolo di un nuovo articolo. Al submit del form, la lista si aggiorna mostrando il nuovo articolo.

**Concetti chiave:** controlled component, gestione dell'evento `onSubmit`, aggiornamento dello state con spread operator.

---

## Bonus

- **Cancellazione:** possibilità di eliminare ciascun articolo tramite un'icona.
- **Modifica:** funzionalità per modificare il titolo di un articolo esistente.
- **Componenti multipli:** suddivisione del lavoro in più componenti React.

---
